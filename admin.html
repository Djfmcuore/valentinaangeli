Gestione foto dello studio

1. Accedi con il tuo token GitHub

Il token non viene salvato da nessuna parte: resta solo nella memoria di questa scheda del browser finché non la chiudi.

2. Carica una nuova foto

Formati consigliati: JPG o PNG, meglio se già ridimensionate (max ~2000px di lato) per non appesantire il sito.

3. Foto attualmente online

Clicca "Elimina" per rimuovere una foto dal sito.
